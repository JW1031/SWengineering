## 프로젝트 정보
* 주제 - 셰어택시-택시 동승 어플리케이션

* Objective: 택시 동승자를 쉽게 찾기 위한 소프트웨어

* Motivation: 버스를 타기 불편한 출퇴근이나 등, 하교 같은 시간대에 사람들은 택시를 많이 이용한다. 하지만 부족한 택시 수와 금전적, 시간적인 문제로 인해 불편함을 호소하는 사람들이 많다. 이를 해결하고자 이 프로젝트를 계획하게 되었다.
